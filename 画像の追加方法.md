# 📸 ギャラリーに画像を追加する方法（ねねさん用）

このギャラリーは **GitHub に画像をアップロードするだけ** で自動更新されます。
パソコン作業やアプリのインストールは不要。ブラウザだけでできます。

公開ページ: https://lun3-2026.github.io/nene-ig-48b8f2048e/

---

## はじめての準備（最初の1回だけ）

1. GitHub の無料アカウントを作る → https://github.com/signup
2. アカウント名（ユーザー名）を VOLO に伝える
3. VOLO がこのリポジトリに「共同編集者」として招待します
4. 届いた招待メールの **Accept（承諾）** を押す

これで準備完了です。

---

## 画像を追加する手順（毎回これだけ）

1. ここを開く 👉 **https://github.com/lun3-2026/nene-ig-48b8f2048e/tree/main/images**
2. 右上の **「Add file ▾」→「Upload files」** を押す
3. 追加したい画像を**ドラッグ＆ドロップ**（複数まとめてOK）
4. 下までスクロールして緑の **「Commit changes」** ボタンを押す
5. **1〜2分待つ** → ギャラリーに自動で反映されます ✨

> 追加した画像は最初は **「アンケート」** カテゴリに入ります。
> もし「その他」に入れたい画像があれば、VOLO に「○○番は その他」と伝えてください（または下の categories.json を編集）。

---

## 画像を削除したいとき

1. `images/` フォルダで消したい画像をクリック → 右上ゴミ箱マーク → Commit
2. 自動でギャラリーからも消えます

---

## カテゴリ（アンケート / その他）を変えたいとき ※上級者向け

`categories.json` というファイルに「ファイル名: poll（アンケート）/ other（その他）」が書いてあります。
ここを書き換えてCommitすると分類が変わります。難しければ VOLO に依頼してください。

---

## 仕組み（参考）

- `images/` フォルダ = 画像の置き場（ここが正）
- `categories.json` = どの画像が「アンケート / その他」かの一覧
- 画像をアップ/削除すると **GitHub Actions** が自動で `gallery.html` を作り直し、公開ページに反映します
